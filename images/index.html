The Three Main Image Formats

PNG

The first type is PNG which stands for Portable Network Graphics. PNG is a lossless image compression format and is commonly used for application graphics. The PNG format supports an alpha channel, or the RGBA color space. PNG is good for storing line drawings, text and iconic graphics at a small file size.

JPEG

JPEG stands for Joint Photographic Experts Group and is used in image file formats. It has two subformats, Exif and JFIF. Exif is mostly used in digital cameras and photographic equipment and JFIF is used on the web. JPEG is is a commonly used on the web because it is compressedwhich makes it useful for storing photographs at a smaller size than other formats.
